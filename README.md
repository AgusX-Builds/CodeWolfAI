# 🐺 CodeWolf AI

CodeWolf AI is a Python application that teaches kids how to code in a fun, game-like way.

## 🌟 Features

- 👤 Create an account
- 🔑 Login system
- 📚 Interactive Python lessons
- 🧠 Python placement quiz
- 💻 Wolf Lab (write and run Python)
- ⭐ XP system
- 🪙 Coins
- 🏆 Achievements
- 💾 Save progress
- 👤 Profile page
